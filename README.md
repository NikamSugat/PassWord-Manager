<h1>Password Manager</h1>

<h2>Overview</h2>
<p>This is a simple password manager written in Bash. It provides a command-line interface for managing passwords. The script allows users to add new passwords, retrieve existing passwords, open specific URLs, and exit the password manager.</p>

<h2>How It Works</h2>
<ul>
    <li><strong>Add a New Password:</strong> Allows users to add a new password. Users can specify the password's name, length, and whether to include uppercase letters, lowercase letters, numbers, and special characters.</li>
    <li><strong>Retrieve a Password:</strong> Lets users retrieve a previously added password by entering the password's name.</li>
    <li><strong>Open URL:</strong> Users can open specific URLs related to the creator's profiles (LinkedIn, Gmail, TryHackMe, and TryHackMe Badge) in their default web browser.</li>
    <li><strong>Exit:</strong> Allows users to exit the password manager.</li>
</ul>
